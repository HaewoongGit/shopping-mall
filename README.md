# shopping-mall 프로젝트
## 프로젝트 개요
온라인 쇼핑몰 프로젝트입니다. 회원가입, 상품 등록, 결제, 리뷰 등 일상적인 온라인 쇼핑물의 기능을 갖추고 있습니다. vue.js, nestJS, graphQL을 활용하여 프론트부터 백엔드 그리고 배포까지 스스로 완결된 프로젝트를 만들어보기 위해서 진행한 프로젝트입니다.

## 프로젝트 기간
2023년 3월 ~ 2023년 7월
